* 实现所有类型的盒子

            * 双边卡扣

            * 外卡扣

            * 柔性卡扣（Living Snap）

            * 金属销轴铰链
            * 分体铰链
            * 双铰链
            * 全长钢琴铰链（Piano Hinge）
            * 阻尼铰链

        * 螺丝
            * 自攻螺丝

            * 内六角螺丝
            * 四角螺柱固定

        * 插销（Pin）
            * 直插销
